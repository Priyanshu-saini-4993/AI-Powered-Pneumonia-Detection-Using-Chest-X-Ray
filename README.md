# AI-Powered-Pneumonia-Detection-Using-Chest-X-Ray
This project develops an AI-based Pneumonia Detection System using deep learning techniques to analyze chest X-ray images. The main objective is to classify X-ray images into two categories: Normal and Pneumonia

⚠️ Model file is not included due to large size.
Download trained model here: https://drive.google.com/file/d/1ggAfpt6Ph8HcsyGh9M8us4nsbHoZ76LM/view?usp=drive_link

Download dataset from here :- https://www.google.com/url?sa=t&source=web&rct=j&opi=89978449&url=https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia&ved=2ahUKEwjHk4jGruySAxXC2DgGHXZ-DFAQFnoECAQQAQ&usg=AOvVaw0W89tn_oarz-tKbs1XrtOX

Features 
- Deep Learning based Pneumonia Detection
- Chest X-ray Image Classification
- CNN Model Implementation
- Real-time Prediction

#Tech Stack
- Python
- TensorFlow / Keras
- Flask
- NumPy, Pandas, Matplotlib

  
#How to Run
1. Clone the repository
2. Install requirements:
   pip install -r requirements.txt
3. Download trained model from the link provided
4. Run:
   python app.py
